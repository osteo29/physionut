# 📋 QUICK REFERENCE - بطاقة سريعة

## في 20 دقيقة فقط:

```
Step 1: 📄 افتح supabase/legacy/SUPABASE_SQL_CLEAN.sql
        └─ انسخ كل محتويات (Ctrl+A → Ctrl+C)

Step 2: 🌐 اذهب https://supabase.com/dashboard
        └─ SQL Editor → New Query
        └─ احذف الكود القديم
        └─ الصق (Ctrl+V)
        └─ اضغط Run ✓

Step 3: 💻 اذهب المجلد وامسح:
        npm run dev

Step 4: 🔗 افتح http://localhost:5173/admin/injuries

Step 5: 📥 في Console (F12):
        import { migrateAllInjuriesToSupabase } from './src/utils/dataMigration.ts';
        await migrateAllInjuriesToSupabase();

Step 6: 🎉 اضغط F5 وشوف 95 إصابة!
```
